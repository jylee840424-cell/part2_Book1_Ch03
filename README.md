# HTML · CSS · JavaScript 수업 학습 기록

수업 시간에 **HTML, CSS, JavaScript** 기초부터 실습까지 학습한 내용을 정리한 레포지토리입니다.  
기본 문법 이해뿐 아니라, 작은 단위의 UI 구현과 기능 추가를 통해 **웹 페이지를 직접 만들어보는 방식**으로 학습했습니다.

---

## 학습 목표
- HTML로 웹 페이지 구조를 만들 수 있다.
- CSS로 레이아웃/스타일을 적용할 수 있다.
- JavaScript로 기본적인 동작(이벤트/DOM 조작)을 구현할 수 있다.
- 간단한 웹 페이지를 처음부터 끝까지 만들어 GitHub에 관리할 수 있다.

---

## 학습 내용
### 1) HTML
- 기본 태그(heading, paragraph, list, img, a 등)
- 폼 요소(input, button 등)
- 시맨틱 태그(header, nav, main, section, footer)
- 페이지 구조 설계 연습

### 2) CSS
- 선택자/우선순위, 박스 모델
- Flexbox, Grid 기반 레이아웃
- 폰트/색상/여백/정렬 등 스타일링
- 반응형 기본 개념(미디어 쿼리)

### 3) JavaScript
- 변수/자료형/조건문/반복문
- 함수, 배열/객체 기초
- DOM 선택 및 조작
- 이벤트 처리(click, input 등)
- 간단한 인터랙션 구현(토글, 폼 검증 등)
